## Turtle Race Game 🐢🏁

This project is a fun simulation of a turtle race built using Python's turtle graphics module. 

The user places a bet on which colored turtle will win, and the turtles race across the screen with randomized movement until a winner is declared.

---

## Features

1. Graphical race simulation using turtle module
2. User betting system using input dialog box
3. Randomized turtle movement for unpredictable race outcomes
4. Automatic winner detection
5. Clean positioning of turtles using loops
6. Includes additional keyboard listener movement practice (commented section for future use)

---

## How the Game Works

Six turtles appear on the screen in different colors.

The user is prompted to place a bet by entering a turtle color.

Each turtle moves forward randomly during the race.

The race continues until one turtle crosses the finish line.

The program announces whether the user won or lost.

---

## How to Run the Project

Make sure Python is installed on your system.

Run:

python main.py
